#  SocialProfile – UI/UX Prototype

SocialProfile is a self-initiated practice project focused on designing a modern social media profile interface.

The goal was simple:  
Practice layout composition, visual hierarchy, and UI structuring by recreating and experimenting with a profile view similar to those found in social platforms.

This is not a production system — it is a design and layout exploration.
